The project is based on [Zuluaga's Tutorial](https://www.youtube.com/playlist?list=PLuEZQoW9bRnTKcse75pub6T3-pi5M7NGy) with the following changes:

### OnSale.Web

* NET Core 5.0 instead of NET Core 2.2
